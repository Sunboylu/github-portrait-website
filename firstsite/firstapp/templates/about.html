用户画像展示问题

怎么度量开发者开发能力?

首先设计开发者的五个维度：贡献度（commit）、活跃度（issue）、影响力（follower）、项目经验（repos）、语言能力（language）

然后主要是利用下面的加权公式:

0.05*issue + 0.15*commit + 0.7*follower + 0.05*repos + 0.05*language

如何得到数据?

用户画像数据来源方式共有两种:
  • GitHub API
  • GHtorrent

开发用户画像意义？

设计实现的面向群体化协同开发的用户画像系统，可以帮助了解开发者，猜测开发者的真实需求和潜在需求，挖掘潜在的开发者群体，为后期的软件工程任务推荐合适的开发者，也为开发者推荐合适的软件开发任务。
